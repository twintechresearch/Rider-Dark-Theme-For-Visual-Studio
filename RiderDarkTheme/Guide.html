Installation steps:

  1. Download and install the theme

  2. Go to https://www.jetbrains.com/lp/mono/ and download the fonts

  3. Extract the fonts and right click install JetBrainsMono-Regular.ttf font

  4. Go in Visual Studio to tools -> options -> fonts and colors and set the font

You have to change to size 11 e.g. then hit ok, repeat the step and set 10 as font size and hit ok. (strange behavior of visual studio)

  1. Optional download https://marketplace.visualstudio.com/items?itemName=OmarRwemi.LinePress and after install setup line spacing to 15% or 20%

The color scheme is far for complete for any additional help please Contribute with a pull request https://github.com/twintechresearch/Rider-Dark-Theme-For-Visual-Studio
